# Literature Synthesis — Project 2: Algebraic Statistics for Phylogenetics

## 1. The algebraic statistics dictionary

Classical phylogenetics asks: from observed DNA sequences at the leaves
of an unknown tree, recover the tree. **Algebraic statistics** turns
this into a geometry question. Each model of sequence evolution is a
polynomial map

  φ_T : Θ ⊂ ℝ^N → ℝ^L,    L = κ^n  (κ = alphabet size, n = # leaves)

from model parameters (root distribution + edge transition matrices)
to joint site-pattern probabilities at the leaves. The **phylogenetic
variety** V(T) = Zariski-closure(φ_T(Θ)) is an algebraic variety in
ℙ^{L−1}; the **phylogenetic ideal** a_T of polynomials vanishing on
V(T) gives the **invariants**. Different tree topologies → different
varieties → polynomial tests for topology.

This dictionary was proposed by Lake (1987) and Cavender–Felsenstein
(1987) but essentially dormant until the mid-2000s work of
Sturmfels–Sullivant (toric ideals for group-based models) and
Allman–Rhodes (general Markov model), enabled by computer algebra.

## 2. General Markov model (GMM) on a tree

### 2.1 Parameterization (Allman–Rhodes 2008 §2)

On a rooted binary tree T^r with n leaves and |E| = 2n − 3 edges:
* root distribution **π_r** ∈ Δ^{κ−1}
* per edge e: **M_e** ∈ ℝ^{κ×κ} row-stochastic (a general Markov
  transition matrix, no additional structure)

Joint distribution:
  P(i_1,…,i_n) = Σ_{histories b ∈ [κ]^{2n−2}} π_r(b_r) · ∏_e M_e(b_s(e), b_f(e))

Equivalent inductive definition via cherries:
* 2-taxon tree: P = diag(π) · M
* recurse: fold off one cherry at a time using tensor contractions.

### 2.2 The phylogenetic variety

V(T) = closure of image of φ_r in ℙ^{L−1}. Key facts:
* V(T) is irreducible (kernel of polynomial map is prime).
* V(T) sits on the trivial-invariant hyperplane Σ P(i) − 1 = 0.
* dim V(T) = N − 1 where N = (κ−1) + |E|·κ·(κ−1) (generic case).

### 2.3 Theorem 4 of Allman–Rhodes (2008)

For the **2-state GMM on a binary n-taxon tree**, the phylogenetic
ideal a_T is generated by **3×3 minors of the 2-dimensional edge
flattenings**. This proved Pachter–Sturmfels Conjecture 13.

For κ = 4 (DNA) and larger κ, the analogous statement reduces to
finding defining polynomials for the **3-leaf tree**.

## 3. Edge flattenings and rank ≤ κ

### 3.1 Definition

Any split A|B of the leaf set gives a matrix unfolding of the tensor
P ∈ ℝ^{κ^n}:
  F_{A|B} ∈ ℝ^{κ^{|A|} × κ^{|B|}}
indexed by states at leaves in A (rows) and states at leaves in B
(columns).

### 3.2 The key theorem (Allman–Rhodes, Eriksson)

If A|B corresponds to an edge of T, then
  **rank(F_{A|B}) ≤ κ**
(generically equal). Conversely, a *random* split that is not induced
by an edge has rank up to κ^{min(|A|,|B|)}.

So **the tree topology is recoverable from rank tests on flattenings**,
provided we have accurate estimates of the joint distribution P (from
aligned sequences).

### 3.3 SVDQuartets (Chifman–Kubatko 2014)

For 4 leaves there are 3 unrooted topologies {12|34, 13|24, 14|23}.
Flatten P along each; the true topology has the lowest "SVD score"
  S_{A|B} = Σ_{i>κ} σ_i²
where σ_1 ≥ σ_2 ≥ ⋯ are singular values of F_{A|B}. Extended to n
taxa by scoring all (n choose 4) quartets and assembling a supertree.

Proven statistically consistent under the multi-species coalescent.
Implemented in PAUP*.

## 4. Group-based models — the Fourier miracle

### 4.1 Group-based models

Jukes–Cantor (JC69), Kimura K80 / K2P, and K3ST are group-based:
their Markov matrices are invariant under a finite group Γ acting on
the state space.
* JC69: Γ = ℤ_2 × ℤ_2 (or S_4 for full symmetry), all bases equivalent
* K2P: Γ = ℤ_2 × ℤ_2, transitions vs transversions distinct
* K3ST: Γ = ℤ_2 × ℤ_2, all three off-diagonal rates distinct

### 4.2 Hadamard / discrete Fourier coordinates (Sturmfels–Sullivant 2005)

Apply the discrete Fourier transform to each leaf coordinate. In these
coordinates, the Markov matrices become diagonal (eigendecomposition
aligned with the group's character table). The resulting map is
**monomial**: the variety becomes a **toric variety**, and its ideal
is generated by **binomials**.

Sturmfels–Sullivant compute Gröbner bases for the toric ideal of each
group-based model; these are the invariants biologists actually use
for discrimination.

## 5. Identifiability (Tensor viewpoint; Kruskal)

The joint distribution at an internal node with two children s, t on
a 3-leaf tree factors as

  P_{ijk} = Σ_h π_h · M₁(h,i) · M₂(h,j) · M₃(h,k)

This is a **rank-κ tensor decomposition**. **Kruskal's theorem**
(1977) gives sufficient conditions (k-rank sums ≥ 2·rank + 2) for
uniqueness of the decomposition → proves identifiability of GMM model
parameters (Allman–Ané–Rhodes 2008).

## 6. Felsenstein pruning — classical likelihood

Given a tree T and an observed site pattern x = (x_1,…,x_n) at leaves,
P(x | T, θ) is computed in O(n · κ²) by postorder tree traversal:
* Initialize: L_leaf(x_leaf) = 1
* At internal v with children c₁, c₂:
  L_v(i) = [Σ_j M_{v→c₁}(i,j) · L_{c₁}(j)] · [Σ_j M_{v→c₂}(i,j) · L_{c₂}(j)]
* At root: P(x|θ) = Σ_i π(i) · L_root(i)

The full joint tensor P ∈ ℝ^{κ^n} is computable site-by-site via
Felsenstein (O(n·κ²) per site × κ^n sites) for validation — or
analytically via the inductive tensor definition of φ_r.

## 8. Practical deliverables of this project

Given DNA sequence data at leaves:
1. Estimate site-pattern frequencies P̂ ∈ ℝ^{κ^n}.
2. For each candidate split A|B: compute flattening F̂, measure
   rank deficiency via smallest singular values.
3. Pick the quartet topology minimizing SVD score (Chifman–Kubatko).
4. For benchmarking: simulate data from a known tree and verify
   topology is recovered.
5. For group-based models: evaluate Fourier-coordinate binomials
   (Sturmfels–Sullivant invariants) on estimated frequencies.
